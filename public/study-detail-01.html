變數宣告

let, const & var

Back

1. ES6 新增了兩種宣告變數的方式：let & const

2. let 宣告變數，宣告後可以重新賦值，宣告時可以不賦值

let age;
age = 23;
console.log(age);// 23
age = 24;
console.log(age);// 24

3. const 宣告常數，宣告後不可以重新賦值，宣告時不可以不賦值

const age;// error

const age = 23;
console.log(age);// 23
age = 24;// error

4. let & const 的作用域為 block scope，也就是 { } 中形成的 block，var 的作用域則是 function scope

5. 因為 let & const 的作用域為 block scope，所以 IIFE 的寫法可以簡化為 { ...your code... }

6. let & const 不能先使用後宣告

console.log(age);// undefined
var age = 23;

console.log(age);// error: not defined
const age = 23;

7. var, let & const 的使用取捨有兩派說法：說法 1，預設使用 const，若有需要更新重新賦值的變數再用 let，在 ES6 環境下不要使用 var，資料來源。說法 2，在大型的 scope 下依然使用 var，let & const 只用在小型的 scope。要選擇哪一種就看個人了
