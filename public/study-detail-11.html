Classes 建立物件

Back

1. ES6 中除了使用傳統物件建構函式建立物件，還可以使用 Classes

            // 傳統寫法
            function Cast(name, gang) {
              this.name = name;
              this.gang = gang;
            }

            Cast.prototype.announce = function () {
              console.log(`吾乃`);
            };

            const cast1 = new Cast('令狐沖', '華山');
            cast1.announce();// 吾乃華山令狐沖
          
            // class 寫法
            class Cast {
              constructor(name, gang) {
                this.name = name;
                this.gang = gang;
              }// 不用逗點
              announce() {
                console.log(`吾乃`);
              }
            }

2. 用 extends 關鍵字建立子 class，可繼承父 class 的屬性及方法

            class Support extends Cast {
