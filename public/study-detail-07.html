1. Array.from() & Array.of()

// Array.from()
const items = document.querySelectorAll('li');// 選取的結果是類陣列的 nodelist，並沒有陣列完整的 methods
const itemsArr = Array.from(items);// Array.from() 將 items 轉為完整的陣列存入 itemsArr

// Array.of()
const ages = Array.of(11, 22, 33, 44);// [11, 22, 33, 44]，將傳入的參數轉為陣列

2. someArray.find(callback()) 尋找符合條件的陣列內容，若條件成立，回傳該值

3. someArray.findIndex(callback()) 尋找符合條件的陣列內容索引值，若條件成立，回傳該索引值

4. someArray.some(callback()) 檢查陣列中是否有任意一個符合條件的元素，回傳布林值

5. someArray.1. every(callback()) 檢查陣列中所有元素是否全部符合條件，回傳布林值
